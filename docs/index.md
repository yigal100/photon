## 📝 Contribution Protocols

**Standard Issue Tool:** **Obsidian** is the recommended interface for accessing and modifying the archives to ensure full compatibility with the knowledge graph.

When submitting updates to the archives:

1.  **Language Design**: Updates to the language specification reside in `docs/language/`.
2.  **Engine Specs**: Compiler implementation details reside in `docs/contributing/`.
3.  **Linking**: Maintain the knowledge graph using `[[WikiLinks]]`.
4.  **Examples**: Theoretical concepts must be accompanied by concrete examples.
